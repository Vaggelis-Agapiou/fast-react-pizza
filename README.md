# Fast React Pizza
[![License: MIT](https://img.shields.io/github/license/Vaggelis-Agapiou/fast-react-pizza)](https://opensource.org/licenses/MIT)

A fast, modern pizza ordering dashboard built with React, Vite, Redux Toolkit, React Router, and Tailwind CSS. Users can browse the pizza menu, manage a cart, place orders, search orders by number, and track delivery status.

---

## Tech Stack

- **React** - popular component-based frontend framework
- **Redux Toolkit** - global state management for cart and user data
- **React Router DOM** - client-side routing between pages
- **Tailwind CSS** - utility-first styling for responsive UI
- **Vite** - fast frontend build tool
- **JavaScript (ES6+)** - app logic and state handling

---

## Features

- Enter your name and start ordering
- Browse pizza menu loaded from remote API
- Add pizzas to cart and adjust quantities
- View cart summary with total quantity and price
- Place orders with phone validation
- Auto-fill delivery address via geolocation and reverse geocoding
- Search orders by order number
- View order status, delivery estimate, and priority state
- Mark existing orders as high priority

## Getting Started

### Prerequisites

- Node.js `18+`
- npm `9+`

---

## Installation

1. Clone the repo:

```bash
git clone https://github.com/Vaggelis-Agapiou/fast-react-pizza
```

2. Install dependencies:
```bash
    npm install
```

3. Start the development server:
```bash 
    npm run dev
```
